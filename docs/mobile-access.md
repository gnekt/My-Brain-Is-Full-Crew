# Using the Crew from Your Phone

> Mobile control is not yet supported in the Codex-first edition of the Crew.

---

## Current status

The old mobile workflow depended on host-specific remote-control features that belonged to the Claude runtime. Those assumptions do not carry over to the Codex-first runtime.

Right now, the Codex edition supports:

- local workspace use on the same machine as your vault
- installed runtime assets under `.codex/`
- dispatcher, agents, and skills running locally through Codex

Right now, the Codex edition does **not** provide:

- phone remote control
- browser relay sessions
- a Codex-native mobile workflow

## What to do instead

For now, use the Crew from the same computer where your vault lives.

If you need quick capture while away from your desk:

1. capture the raw note somewhere convenient
2. return to your vault machine
3. paste it into Codex for normal Crew processing

## Future direction

If a Codex-native remote workflow becomes viable later, this document should be replaced with concrete setup instructions. Until then, mobile access should be treated as unsupported rather than partially working.
